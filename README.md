# modern-web-portfolio

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![Status](https://img.shields.io/badge/Status-Active-success)]()

Modern, responsive ve minimalist bir kişisel portföy sitesi. Web geliştirme yeteneklerimi, projelerimi ve sunduğum hizmetleri sergilemek amacıyla geliştirilmiştir.

---

## 📖 Hakkında
Bu proje, temiz kod prensipleri ve modern CSS teknikleri kullanılarak hazırlanmıştır. Kullanıcılara akıcı bir deneyim sunmak için optimize edilmiş olup, cihaz bağımsız (mobile-first) bir tasarım anlayışıyla kurgulanmıştır.

## 🛠️ Teknolojiler
Proje geliştirilirken kullanılan ana teknolojiler:
* **HTML5:** Anlamsal ve erişilebilir yapı.
* **CSS3:** Modern layout teknikleri (Flexbox/Grid), animasyonlar ve responsive tasarım.
* **JavaScript:** DOM manipülasyonu ve interaktif elementler.

## 🚀 Öne Çıkan Özellikler
* **Responsive Layout:** Tüm ekran boyutlarında kusursuz görünüm.
* **Modern UI/UX:** Kullanıcı odaklı, estetik ve profesyonel arayüz tasarımı.
* **Performans Odaklı:** Minimum bağımlılık, maksimum hız.
* **GitHub Entegrasyonu:** Açık kaynaklı projelerin kolayca incelenebilmesi için optimize edilmiş bölümler.
